Elm Blog Tutorial
=================

This is a tutorial code following the [Single-Page Web Apps in Elm](https://www.linkedin.com/pulse/single-page-web-apps-elm-part-one-getting-started-new-kevin-greene) series by Kevin Greene.

Reference [repository](https://github.com/kevinbgreene/elm-tutorial)

Make it run with `elm-reactor` or:

```
elm-package install
npm run build
open public/index.html
```
